# 📖 AI-Generated Interactive Story (Beta)

An experimental narrative game built with **React** + **Node.js proxy backend**, where each scene is generated by a large language model (LLM).
Player choices directly shape the unfolding story, with dynamic memory, evolving characters, and branching paths.

---

## ✨ Features

* **Dynamic Storytelling**
  Stories are generated scene-by-scene using LLMs. Each turn includes:

  * A vivid story segment
  * Four unique choices for the player
  * Updated character traits, roles, and emotional states
  * A concise scene summary for memory tracking

* **Interactive Choices**
  Player choices directly alter the narrative and influence characters’ emotions and relationships.

* **Character Memory**
  Characters evolve across scenes, storing traits, dialogue, and relationship history.

* **Game Save/Load**
  Save your adventure to one of three slots and resume later.

* **Frontend/Backend Separation**

  * **React frontend** for UI
  * **Express proxy server** to call OpenAI safely without exposing your API key

* **Styling**
  Cozy UI with **Tailwind CSS**, animated text transitions, and background imagery.

## 🚀 Getting Started (Local Development)

### 1. Clone the repository

```bash
git clone https://github.com/qmanhbeo/AI-generated-interactive-story-beta.git
cd AI-generated-interactive-story-beta
```

---

### 2. Set up the backend (API proxy)

The backend lives in `/server`. It hides your LLM API key so it’s never exposed in the browser.

```bash
cd server
npm install
```

Create a `.env` file inside `/server`:

```env
OPENAI_API_KEY=sk-...your key...
UPSTREAM_URL=https://api.openai.com/v1/chat/completions
MODEL=gpt-4o-mini
PORT=5174
```

Start the backend:

```bash
npm run dev
```

You should see:

```
[api-proxy] listening on http://localhost:5174
```

---

### 3. Configure the frontend

In the project root, create a `.env` file with safe, non-secret variables:

```env
REACT_APP_API_BASE=http://localhost:5174/api
REACT_APP_LLM_MODEL=gpt-4o-mini
```

> ⚠️ Do **not** put your real API key here. That stays only in `/server/.env`.

---

### 4. Install frontend dependencies

```bash
npm install
```

---

### 5. Run the frontend dev server

```bash
npm start
```

By default it runs at:

* Local: [http://localhost:3000](http://localhost:3000)
* Network: `http://<your-ip>:3000`

---

### 6. Play the game 🎮

1. Choose your **genre, protagonist, tone, setting** at the start screen.
2. Read the first generated story segment.
3. Pick from four choices to shape the narrative.
4. Watch characters evolve and the world respond.
5. Save your game anytime.

## 🛠 Tech Stack

* **Frontend**: React (CRA), Tailwind CSS v3, React Hooks
* **Backend**: Express, CORS, node-fetch, zod (validation)
* **AI Integration**: OpenAI API (via server proxy)
* **State & Memory**: Custom managers for story, choices, characters, emotions
* **Testing**: CRA Jest setup (expandable)

---

## 📂 Project Structure

```
/components        → UI components (GameScreen, StartScreen, etc.)
/components/GameScreenComponents → ChoiceGrid, HeaderBar, StoryDisplay
/utils             → Core logic (AI-chat, buildUnifiedPrompt, memory, parser)
/server            → Express proxy to OpenAI (safe API key storage)
/images            → Backgrounds and UI assets
```

---

## 📝 Notes

* Requires **Node.js ≥ 18** and npm.
* If dependencies mismatch, run `npm ci` for reproducible installs.
* Backend is required — the game will not work if you only run `npm start` in root.
* Secrets (`server/.env`) are **ignored by git**. Never commit your real API key.
